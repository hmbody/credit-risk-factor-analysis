# 房地产行业信用风险因子分析

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

> 基于 A 股 71 家房地产上市公司 2020-2024 年面板数据，运用双向固定效应模型，识别影响企业信用风险的关键财务因子。
> 
> **核心结论：ROA 和毛利率是信用风险最显著的两个保护因子（P<0.01），而资产负债率本身对偿债能力的影响不显著——关键不在"借了多少"，在"赚了多少"。**

---

| 维度 | 说明 |
|------|------|
| 来源 | Tushare 金融数据库 |
| 样本 | A 股房地产行业上市公司（非 ST，2019 年前上市） |
| 规模 | **71 家公司 × 5 年 = 325 条观测** |
| 时间 | 2020 — 2024 年 |

## 核心发现

### 基准回归（双向固定效应模型）

**因变量**：利息覆盖率（EBIT / 利息支出）——值越高，信用风险越低

| 变量 | 系数 | P 值 | 显著性 | 经济含义 |
|------|------|------|:--:|------|
| **ROA** | +1.446 | < 0.001 | ★★★ | ROA 每提高 1 个百分点，利息覆盖率提高 1.45 |
| **毛利率** | +0.294 | < 0.001 | ★★★ | 毛利率每提高 1 个百分点，利息覆盖率提高 0.29 |
| **流动比率** | +3.629 | 0.011 | ★★ | 短期偿债缓冲有独立保护作用 |
| 资产周转率 | +13.687 | 0.065 | ★ | 运营效率有边际保护作用 |
| 资产负债率 | — | 0.764 | 不显著 | 杠杆本身不直接决定信用风险 |
| ROE | — | 0.635 | 不显著 | 被 ROA 和杠杆效应吸收 |

> R² = 0.259 &nbsp;|&nbsp; F = 13.82 (P < 0.001) &nbsp;|&nbsp; N = 325

**关键解读**：资产负债率不显著——这意味着高杠杆本身不是问题，问题在于**借来的钱有没有转化为盈利能力**。两家同样负债率 70% 的公司，ROA 高的那家信用风险显著更低。

### 异质性分析（按资产负债率分组）

按杠杆水平将公司分为低杠杆组和高杠杆组，分组回归后结论一致——ROA 和毛利率在两个组中均为显著保护因子，结果稳健。

### 稳健性检验

1. **替换因变量**：用 ROA 替代利息覆盖率重新回归，结论方向一致
2. **缩尾处理验证**：未缩尾数据结果与主回归一致
